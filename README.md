# Spring Data JPA Learning Project

## 📌 Overview

This project is created to learn and understand the core concepts of **Spring Data JPA**. It demonstrates how to interact with a database using JPA repositories and perform **CRUD operations, entity mapping, and custom queries** in a Spring-based application.

The goal of this project is to reduce boilerplate code and simplify database access using **Spring Data JPA features such as repositories, annotations, and query methods**.

---

## 🚀 Features

* Perform **CRUD Operations**
* **Entity Mapping** using JPA annotations
* **Custom Queries** using JPQL / HQL
* **Repository Interfaces**
* **Pagination and Sorting**
* Database interaction using **Spring Data JPA**

---

## 🛠️ Technologies Used
